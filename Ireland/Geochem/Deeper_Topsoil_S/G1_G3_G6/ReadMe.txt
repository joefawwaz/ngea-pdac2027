Title	58xxxxS-65xxxxS_Deeper_Topsoil_Download_v2_2
Purpose	Multi-element geochemical data from deeper topsoil surveying in Ireland, sample media code 'S'.
Version	2.2
Summary	Concentration geochemical data for analytes by pH (CaCl2), LOI (at 450°C), XRFS and ICPar analyses on <2 mm fraction deeper soil (up to 0.50 m depth).
Description	Unique sample ID is a combination of site number and sample media code.
	Location co-ordinates data easting and northing are in Irish National Grid.
	Data are censored where below LLD. Blank cells are null (no sample/data available).
	Users are cautioned that this dataset contains XRFS results that have had correction factors applied. Conditioned analytes are: K2O, V, Co, Cu and Yb only.
Revision replaces v2.1
Change log:
	1.  Missidentification of small number of QC samples was corrected. All relevant QC outputs were updated.
	2.  Additonal CRMs (OREAS series) were included in regression calculations.
	3.  Minor updates to reported LLDs and inclusion of note on conditioned data.
