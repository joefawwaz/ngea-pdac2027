Purpose	Multi-element geochemical data from stream sediment surveying in Ireland, sample media code 'C'.
Version	1.1 (replaces 1.0)
Summary	Geochemical data for analytes by XRFS and FA ICP-MS analyses on <150 ”m fraction low-order stream sediment.
Description	Unique sample ID is a combination of site number and sample media code.
	Location co-ordinates data easting and northing are in Irish National Grid.
	Data are censored where below LLD. Blank cells are null (no sample/data available).
Release notes	 Rounding of the reported values was updated.
Supporting documentation 	Knights, K.V. Tellus geochemical survey: stream sediment data from the border, north, west and southeast of Ireland. Geological Survey Ireland, 2020. 
URL	https://www.gsi.ie/en-ie/data-and-maps/Pages/Geochemistry.aspx
Author	M. Szpak. Geological Survey Ireland | Tellus
Date	March 2020
Contact	tellus@gsi.ie

Folder 'C_Stream_sediment_geochemistry' contains Excel spreadsheet with tabulated data for all variables with heading containing concentration unit information. Classes are symbolised to 3 significant figures and are based on percentile calculations as standard of 0, 5, 10, 15, 25, 50, 75, 90, 95, 99 and 100. Data are projected in Irish National Grid (GCS_TM65). Raster files available upon request.
