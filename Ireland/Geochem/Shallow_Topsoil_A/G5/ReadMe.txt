Title	6117xxA-6174xxA_Shallow_Topsoil_Download_v1.1
Purpose	Multi-element geochemical data from shallow topsoil surveying in Ireland, sample media code 'A'.
Version	1.1.
Summary	Concentration geochemical data for analytes by pH (CaCl2), LOI (at 450°C)and ICPar analyses on <2 mm fraction shallow topsoil (up to 0.20 m depth).
Description	Replaces v1.0.
	Unique sample ID is a combination of site number and sample media code.
	Location co-ordinates data easting and northing are in Irish National Grid.
	Data are censored where below LLD. 
	For elements where >80% of observations fall below the accredited LLD, non-accredited, uncensored below-LLD data are also provided (B, Ta, Pd and Pt). 
Release notes	LLDs for pH and LOI added.
Supporting documentation 	Szpak, M.T. Tellus geochemical survey: shallow topsoil data from the midlands of Ireland . Geological Survey Ireland, 2020.  
URL	https://www.gsi.ie/en-ie/data-and-maps/Pages/Geochemistry.aspx
Author	M.T. Szpak & V.Lowe. Geological Survey Ireland | Tellus
Date	October 2020
Revision date	March 2023
