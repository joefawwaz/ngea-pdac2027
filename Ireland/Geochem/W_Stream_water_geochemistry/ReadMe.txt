Title	58xxxxW-60xxxxW_Stream_Water_Download_v1.2
Purpose	Multi-element geochemical data from stream water surveying in Ireland, sample media code 'W'.
Version	1.1 (replaces 1.0)
Summary	Geochemical data for analytes by pH, SEC, total alkalinity, NPOC, IC and ICP-MS on water collected form low-order streams.
Description	Unique sample ID is a combination of site number and sample media code.
	Location co-ordinates data easting and northing are in Irish National Grid.
	In case of non-detects, i.e. data reported by the laboratory as "< LLD", data are censored to half the method LLD. Blank cells are null (no sample/data available).

Folder 'W_Stream_water_geochemistry' contains contains Excel spreadsheet with tabulated data for all variables with heading containing concentration unit information. Classes are symbolised to 3 significant figures and are based on percentile calculations as standard of 0, 5, 10, 15, 25, 50, 75, 90, 95, 99 and 100. Data are projected in Irish National Grid (GCS_TM65). Raster files available upon request.
